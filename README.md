变量名 DISCORD_ACCOUNTS	
，多账号配置（一行一个账号）。


格式：邮箱,密码,2FA纯文本秘钥


(注：如果没有 2FA，请保留最后一个逗号)


user1@gmail.com,Pass123!,ABCD1234EFGH


user2@qq.com,Pass456@,


user3@163.com,Pass789#,IJKL5678MNOP

变量名 GOST_PROXY	
，socks5://用户名:密码@IP地址:端口


变量名 TG_BOT	
，Telegram 机器人的推送配置。


格式：你的ChatID,你的BotToken

注：借鉴了losy-mify大佬的项目进行修改搭建
